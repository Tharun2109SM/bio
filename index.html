🔄 What is a "Mesh"?

A mesh is a closed loop in a circuit — like a window frame. We assign a clockwise current to each loop and call them i₁, i₂, i₃. The goal is to find these currents.


This is called Mesh Analysis (KVL method) — it's the most systematic way to solve multi-loop circuits.

Mesh 1 — i₁
Mesh 2 — i₂ (has current source)
Mesh 3 — i₃

💡 Key Rule — Shared Resistors

When a resistor sits between two mesh loops, the current through it is the difference of the two mesh currents. For example, the 3Ω resistor is shared between i₁ and i₂, so current through it = i₁ − i₂.

✍️ Writing KVL Equations

For each mesh, we apply Kirchhoff's Voltage Law (KVL): "Sum of all voltage drops around a loop = 0". Voltage drop across a resistor = R × (mesh current through it).

1 Mesh 1 Equation — i₁ loop

Going clockwise around Mesh 1. We hit the 7V source, then resistors 1Ω, 3Ω, 2Ω:

KVL — Sum of voltages = 0
+7 − 1(i₁) − 3(i₁ − i₂) − 2(i₁ − i₂) = 0

Expanding and collecting terms:

Simplified — Equation 1
i₁ − 4i₂ + 4i₃ = 7   → (Eq. 1)

2 Mesh 2 — Current Source Trick!

Mesh 2 contains the 7A current source. When a current source is between two meshes, you do NOT write KVL. Instead, you write the supermesh equation directly — the difference of currents = source value:

Current Source Constraint — Equation 3
i₁ − i₃ = 7   → (Eq. 3)

✦ This is the simplest equation — directly from the 7A source sitting between mesh 1 and mesh 3

3 Mesh 3 Equation — i₃ loop

Going clockwise around Mesh 3. Resistors: 3Ω (shared with mesh 1), 2Ω, 1Ω:

KVL — Mesh 3
−3(i₃ − i₁) − 2(i₃) − 1(i₃) = 0
Simplified — Equation 2
−i₁ + 6i₂ − 3i₃ = 0   → (Eq. 2)

📊 The 3 Equations System

i₁ − 4i₂ + 4i₃ = 7      (1)
−i₁ + 6i₂ − 3i₃ = 0    (2)
i₁ − i₃ = 7              (3)

🧮 Solving the System of Equations

We have 3 unknowns: i₁, i₂, i₃. We solve step-by-step using substitution.

A From Equation (3) → Express i₁

Eq (3): i₁ − i₃ = 7
∴ i₁ = i₃ + 7

B Substitute i₁ into Equation (1)

Eq (1): i₁ − 4i₂ + 4i₃ = 7 → substitute i₁ = i₃ + 7
(i₃ + 7) − 4i₂ + 4i₃ = 7
Simplify
5i₃ − 4i₂ + 7 = 7
Result
i₂ = 5i₃ / 4   → (4)

C Substitute into Equation (2) to find i₃

Eq (2): −i₁ + 6i₂ − 3i₃ = 0 → substitute i₁ and i₂
−(i₃+7) + 6(5i₃/4) − 3i₃ = 0
Expand
−i₃ − 7 + 30i₃/4 − 3i₃ = 0
Collect i₃ terms  (−1 + 7.5 − 3 = 3.5)
3.5 × i₃ = 7
✅ Found i₃
i₃ = 7 / 3.5 = 2 A

D Back-substitute to find i₁ and i₂

i₁ = i₃ + 7
i₁ = 2 + 7 = 9 A
i₂ = 5×i₃/4
i₂ = 5×2/4 = 10/4 = 2.5 A

⚡ Finding Voltage Across 3Ω

The 3Ω resistor sits between Mesh 1 and Mesh 2. The current through it is their difference:

Current through 3Ω (shared between i₁ and i₂)
I₃Ω = i₁ − i₂ = 9 − 2.5 = 0.5 A
Ohm's Law: V = I × R
V₃Ω = 0.5 × 3 = 1.5 V
VOLTAGE ACROSS 3Ω RESISTOR
1.5 V
V = I × R = 0.5A × 3Ω = 1.5 V
